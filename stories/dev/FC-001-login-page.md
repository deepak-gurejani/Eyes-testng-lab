# FC-001 — Build Login Page

Type: Story
Role: Developer
Status: Done
Sprint: Sprint 1
Priority: P0
Story Points: 3
Epic: FreshCart Web App MVP

## User Story

As a Developer, I want to build a responsive login page for FreshCart so that customers have a polished first impression and can access their account.

## Acceptance Criteria

- AC-1: Page displays the FreshCart logo (cart icon and brand name) in the header
- AC-2: Page displays the tagline "Fresh groceries, delivered fast"
- AC-3: Email input field is present with placeholder text
- AC-4: Password input field masks characters as dots
- AC-5: Sign In button is full-width and green
- AC-6: Clicking Sign In navigates to the Dashboard page
- AC-7: Footer displays copyright with company name and city
- AC-8: Page is responsive across desktop, tablet, and mobile viewports

## Definition of Done

- HTML and CSS implemented
- Code reviewed via PR
- Merged to develop branch
- Visual regression tests covered by FC-201

## Out of Scope

- Real authentication
- Forgot password flow
- Account creation flow

## Linked Stories

- FC-002 and FC-003 (other pages)
- FC-101 (deployment)
- FC-201 (QA visual coverage)
